Scale Mail is an inventory management puzzle game created for the yearly GMTK game jam, with the theme "Built to Scale." I led the Godot development process, bringing programmer teammates up to speed with the engine and implementing major features such as item grid placement logic, score calculations, item/bag/inventory infrastructure, and smooth UI effects and transitions.

  • Designed a detailed process of defining items using a coordinate system that accounted for different scales and rotations
  • Worked closely with artists to ideate and implement art assets
  • Developed in an expandable way, planning for future use of the systems and tools I created (ex: an editor-exposed bag customization system)
  • Maintained a clean codebase through code management with GitHub and a dedicated Discord server using webhooks for realtime updates to the team
